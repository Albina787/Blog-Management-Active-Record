# Blog Management System (Active Record)

## Description
This project demonstrates the implementation of the Active Record pattern in Java.
The system manages blog articles and comments with CRUD operations.

## Technologies
- Java
- MySQL
- JDBC

## Features
- Create, Read, Update, Delete articles
- Add and delete comments
